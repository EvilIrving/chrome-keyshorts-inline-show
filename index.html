Press and hold. See hotkeys. Release.

A lightweight Chrome extension that shows a hotkey panel when you hold a configurable key. Auto-detects Mac or Windows.

  • 1 No clicks neededHold Right Alt for 0.8 seconds; release to dismiss. Keep your hands on the keyboard.
  • 2 Mac and Windows layoutsAuto-detects your platform. Or lock to one layout in the options.
  • 3 ConfigurableChoose your trigger key (Alt, Shift, Ctrl), hold duration, and platform layout.
  • 4 Zero data collectionNo network calls, no analytics, no tracking. Settings stay in your browser.

How it works

  1. Open any web page (http or https).
  2. Hold the trigger key until the hotkey panel appears.
  3. Release the key to dismiss. To change settings, click the extension icon in your toolbar.

Privacy Policy

Effective date: April 17, 2026

Chrome Hotkeys is a locally-running Chrome extension. It shows a floating hotkey reference panel when you long-press a configurable key on any web page.

1. What we collect

This extension does not collect, upload, or sell any personal information.

  • It stores 3 user preferences locally: trigger key, hold duration, and hotkey layout (Auto / Mac / Windows).
  • These preferences are saved via Chrome's chrome.storage.sync API and only synchronize within your own Chrome browser.
  • The extension does not read passwords, form data, browsing history, page content, chat messages, payment information, or identity information.

2. How we use information

Stored preferences are used only for:

  • Recognizing your chosen trigger key
  • Determining how long you need to hold before the panel appears
  • Showing Mac or Windows hotkey labels

3. Permissions explained

  • storage: used to persist your settings.
  • Content scripts on http://*/* and https://*/*: used to listen for the trigger key and render the hotkey panel on the current page.

4. Data sharing

This extension does not send data to any developer server and does not share user data with third parties.

5. Data storage and security

Settings are stored in Chrome's built-in storage mechanism. The extension has no remote servers, databases, or analytics platforms.

6. Children's privacy

This extension is a general-purpose hotkey reference tool. It contains no data collection logic targeting children.

7. Policy updates

If the extension adds new permissions, network capabilities, or data processing in the future, this page will be updated with a new effective date.

8. Contact

To reach the developer, use the support email listed on the Chrome Web Store extension details page.

Common questions

The hotkey panel doesn't appear when I hold the key.
Make sure you are on an http or https page (not chrome:// or a new tab). Try increasing the hold duration in the options. Also check that your trigger key is not being intercepted by another extension.
How do I change the trigger key or hold duration?
Click the extension icon in the Chrome toolbar to open the options popup. Changes save automatically.
Does this extension work on Edge, Brave, or other Chromium browsers?
Yes. It works in any Chromium-based browser that supports Manifest V3 extensions.
