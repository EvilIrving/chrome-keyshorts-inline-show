# Product

## Register

product

## Users

Chrome users on both Mac and Windows who want quick keyboard hotkey reference. They use this in the flow of browsing — they need answers fast, without friction. English and Simplified Chinese speaking.

## Product Purpose

A Chrome extension that shows a floating keyboard hotkey panel when the user holds a configurable trigger key. This index.html is the extension's public web presence: a landing page with Home, Privacy, and Support tabs.

## Brand Personality

Simple, trustworthy, minimal. The extension collects zero data — the landing page should radiate that same restraint and confidence. No overpromising, no marketing fluff.

## Anti-references

- Chrome Web Store generic listing pages with badge walls and feature grids
- SaaS landing pages with hero metrics, gradient CTAs, and pricing tables
- Over-designed privacy policies that look like legal threats

## Design Principles

- **Show restraint**: Every element must earn its place. Privacy-respecting code deserves a privacy-respecting page.
- **Clarity over persuasion**: This is documentation, not a sales pitch. State facts, don't sell.
- **One surface, three lenses**: Home, Privacy, Support share one clean frame — tabs filter the view, they don't create separate pages.
- **No dark patterns**: No tracking, no newsletter popups, no social proof nudges.

## Accessibility & Inclusion

WCAG 2.1 AA. Support reduced motion. Keyboard-navigable tabs. Clear focus indicators. Readable contrast ratios throughout.
