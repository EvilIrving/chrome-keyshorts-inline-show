Chrome 热键 - 隐私政策

生效日期：2026-04-17

Chrome 热键是一款本地运行的 Chrome 扩展，用于在网页内长按指定按键时显示 Chrome 常用热键面板。

1. 我们收集什么信息

本扩展默认不收集、不上传、不出售任何个人信息。

  • 扩展会在本地保存 3 项用户设置：触发键、长按时长、热键布局（自动 / Mac / Windows）。
  • 这些设置通过 Chrome 提供的 chrome.storage.sync 保存，用于在用户自己的 Chrome 浏览器中同步。
  • 扩展不会读取账号密码、表单内容、浏览历史、网页正文、聊天内容、支付信息或身份信息。

2. 我们如何使用信息

保存的设置仅用于以下功能：

  • 识别你选择的触发键
  • 决定显示热键面板所需的长按时长
  • 决定显示 Mac 版或 Windows 版热键列表

3. 权限说明

  • storage：用于保存用户设置。
  • content_scripts 匹配 http://*/* 和 https://*/*：用于在已打开网页中监听触发键，并在当前页面渲染热键面板。

4. 数据共享

本扩展不向开发者服务器发送数据，也不与第三方共享用户数据。

5. 数据存储与安全

设置数据保存在 Chrome 浏览器提供的存储机制中。扩展自身没有远程服务器、数据库或分析平台。

6. 儿童隐私

本扩展面向一般用户，功能为热键工具。扩展没有专门面向儿童设计的数据收集逻辑。

7. 政策更新

如果扩展未来增加新的权限、联网能力或数据处理逻辑，本页面会同步更新生效日期和说明。

8. 联系方式

如需联系开发者，请在 Chrome Web Store 扩展详情页填写你的开发者邮箱，并将该邮箱同步放在这里。
